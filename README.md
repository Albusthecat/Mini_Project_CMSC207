# 🚀 Personal Web Profile | Kyne De Guzman
**Bridging Finance & Technology through Code and Linguistics**

This project is a responsive personal portfolio built as part of my **CMSC 207 (DCS)** coursework. It highlights my journey as a career shifter and language learner, using a clean, interactive design.

---

## 🌟 Key Features

| Feature | Description |
| :--- | :--- |
| **Dual-Theme** | Toggle between a professional **Dark Mode** (Black/Gray) and a vibrant **Light Mode** (Pink). |
| **Typing Intro** | A dynamic greeting that cycles through five languages (English, Tagalog, Spanish, French, Thai). |
| **Fun Facts** | A random fact generator highlighting my background in finance, my cat Albus, and my language studies. |
| **Active Nav** | Smart navigation that highlights your current location on the page as you scroll. |
| **Responsive** | Mobile-first design using CSS Flexbox and smart image positioning for perfect framing. |

---

## 🛠️ Technical Stack

* **HTML5:** Semantic structure including About, Hobbies, and Contact sections.
* **CSS3:** Utilizes custom properties (Variables), smooth transitions, and Intersection Observer animations.
* **JavaScript:** Pure Vanilla JS handles all interactive logic without heavy external dependencies.
* **Assets:** Optimized photography focused on hobbies like badminton and travel.

---

## 📂 Project Structure

* `index.html`: Core content and semantic markup.
* `style.css`: All styling, theme variables, and responsive media queries.
* `script.js`: Logic for the theme toggle, typing effect, and fun facts.
* `/images`: Curated images of Albus, badminton matches, and family travels.

---

## 📝 Self-Reflection

As a first-time developer, my biggest challenge involved mastering **CSS layouts**. I specifically learned how to use `object-fit` and `object-position` to ensure my photos—especially for my badminton and family sections—looked professional in different container sizes. This project reinforced that the logic I used in **finance** translates perfectly into debugging and structured programming.
