# Personal Web Profile - Kyne De Guzman

A modern, responsive personal web profile built as a mini-project for CMSC 207. This site showcases my journey as a DCS student and language learner, bridging my previous background in finance with my current pursuit of computer science.

## 🚀 Features

- **Dual-Mode Theme:** Includes a sleek black-and-gray Dark Mode and a vibrant pink Light Mode.
- **Interactive Fun Fact Generator:** A customized JavaScript tool that shares facts about my career pivot, language studies, and my cat, Albus.
- **Dynamic Typing Effect:** An animated header that greets visitors in multiple languages (English, Tagalog, Spanish, French, and Thai).
- **Responsive Hobby Layout:** Side-by-side cards that adapt to mobile screens, using advanced CSS object-positioning to ensure portraits and badminton medals are perfectly framed.
- **Seamless Navigation:** Smooth scrolling and active-state highlighting to help users explore my profile effortlessly.

## 🛠️ Technology Stack

- **HTML5:** Structured for accessibility and semantic clarity.
- **CSS3:** Built with custom variables, Flexbox for layout, and Intersection Observer for fade-up animations.
- **JavaScript (Vanilla):** Used for theme toggling, typing animations, and DOM manipulation without external libraries.

## 📂 Project Structure

- `index.html` - The core structure of the profile.
- `style.css` - Custom styles including color variables, responsive design, and animations.
- `script.js` - Interactive logic for scrolling, themes, and animations.
- `/images` - Optimized assets for hobbies and personal photography.

## 💡 Self-Reflection

This project marked my first time building a website from scratch. The most significant learning curve involved mastering CSS `object-fit` and `object-position` to handle portrait photography in horizontal containers. I also learned the importance of clean code documentation, especially when debugging syntax errors across multiple files.
